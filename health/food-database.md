## 📝 使用说明

### 卡路里计算公式
```
总卡路里 = Σ(食物重量 (g) × 单位卡路里 / 100)
```

### 示例计算
```
早餐：
- 米饭 1 碗 (200g) = 200 × 130 / 100 = 260 kcal
- 鸡蛋 1 个 (50g) = 50 × 155 / 100 = 78 kcal
- 牛奶 1 杯 (250ml) = 250 × 61 / 100 = 153 kcal
早餐总计：491 kcal
```

### 16+8 断食判断
```
进食窗口 = 第一餐时间 → 最后一餐时间
断食窗口 = 24h - 进食窗口

16+8 标准：断食窗口 ≥ 16h
```
