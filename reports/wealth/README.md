## 📈 数据分析建议

### 1. 收益率趋势

使用 Excel 图表功能：
- 插入 → 折线图
- 选择三个时间节点的收益率数据
- 分析收益率上升/下降趋势

### 2. 银行排名变化

- 使用条件格式标记排名变化
- 绿色：排名上升
- 红色：排名下降
- 黄色：排名不变

### 3. 期限收益对比

- 对比 3/6/12 个月的收益率差异
- 计算期限溢价（长期 - 短期）
- 评估是否值得锁定长期

### 4. 风险收益分析

- 筛选相同风险等级的产品
- 对比收益率差异
- 选择最优风险收益比

## ⚠️ 投资风险提示

1. **业绩基准≠实际收益**
   - 业绩比较基准是预期收益
   - 不承诺保本保息
   - 实际收益可能高于或低于基准

2. **理财非存款**
   - 银行理财产品不享受存款保险
   - 存在本金损失风险

3. **风险匹配原则**
   - R1(低风险)：保守型投资者
   - R2(中低风险)：稳健型投资者
   - R3(中风险)：平衡型投资者
   - R4(中高风险)：成长型投资者
   - R5(高风险)：进取型投资者

4. **分散投资**
   - 不要将所有资金投入单一产品
   - 分散到不同银行、不同期限

5. **流动性考虑**
   - 理财产品通常有封闭期
   - 提前赎回可能有损失或无法赎回
