## 📋 统一命名格式

### 每日统计文件
```
YYYY-MM-DD-growth-stats.md
YYYY-MM-DD-growth-stats.json
```

**示例：**
- `2026-03-10-growth-stats.md`
- `2026-03-11-growth-stats.md`

### 每周汇总文件
```
YYYY-Www-growth-summary.md
```

**示例：**
- `2026-W11-growth-summary.md` (第 11 周)
- `2026-W12-growth-summary.md`

### 每月汇总文件
```
YYYY-MM-growth-summary.md
```

**示例：**
- `2026-03-growth-summary.md`

## 📝 统计内容

### MD 文件包含：
1. 📋 今日计划（晨间填写）
2. 📊 实际完成（晚间复盘后填充）
3. 📝 产出统计
4. 💡 问题与建议
5. 📈 本周累计
6. 🔗 相关文件链接

### JSON 文件包含：
- 结构化数据（便于程序处理）
- 周次/天数计算
- 12 周计划进度对比

---

## 🔄 生成流程

1. **08:00** - 自动发送晨间计划问卷（飞书）
2. **用户回复** - 填写今日学习计划
3. **21:00** - 自动发送晚间复盘问卷（飞书）
4. **用户回复** - 填写实际完成情况
5. **22:00** - 解析回复，生成统计文件
6. **22:05** - 发送确认反馈给用户

---

## 📊 核心指标

| 维度 | 指标 | 计算方式 |
|------|------|---------|
| **学习时长** | 总分钟数 | OpenClaw + Claude + 视频 |
| **完成率** | 计划完成度 | 实际时长/计划时长 × 100% |
| **质量评分** | 1-5 分 | 用户自评 |
| **连续性** | 连续学习天数 | streak 计算 |
| **产出** | 笔记/实操/作品 | 累计计数 |

## 🎯 12 周计划集成

### 周次计算
- **起始日期：** 2026-03-09
- **当前周次：** Math.ceil((今天 - 起始) / 7)
- **周进度：** (今天 - 起始) % 7 + 1
