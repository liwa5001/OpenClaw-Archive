# 📚 成长堡 - 每日计划与复盘系统 (Castle Six)

**版本：** v1.0  
**生效日期：** 2026-03-10  
**维护者：** 城堡 🏰

---

## 🎯 设计原则

1. **轻量级** - 每日 2-3 分钟完成，避免负担
2. **可执行** - 计划具体到时间段和任务
3. **有反馈** - 复盘后自动生成明日建议
4. **可追踪** - 数据沉淀到成长堡统计文件

## 📊 数据统计

### 每日统计指标

| 维度 | 指标 | 计算方式 |
|------|------|---------|
| **学习时长** | 总分钟数 | OpenClaw + Claude + 视频 |
| **完成率** | 计划完成度 | 实际时长/计划时长 |
| **质量评分** | 1-5 分 | 用户自评 |
| **产出统计** | 笔记/实操/作品 | 累计计数 |
| **连续性** | 连续学习天数 |  streak 计算 |

### 每周统计指标

| 维度 | 指标 | 目标值 |
|------|------|--------|
| 总学习时长 | 分钟/周 | ≥630min (90min×7) |
| 平均质量分 | 1-5 分 | ≥3.5 |
| 完成天数 | 天/周 | ≥5 天 |
| 作品产出 | 个/周 | ≥1 个 |

## 💡 智能建议逻辑

### 基于完成率的建议
```
IF 连续 3 天未完成 → "建议降低每日目标，先建立习惯"
IF 连续 7 天完成 → "恭喜！可以尝试增加挑战"
IF 质量评分<3 → "建议调整学习内容或时间"
```

### 基于时长的建议
```
IF 总时长<30min → "学习时间不足，建议固定时段学习"
IF 总时长>120min → "学习强度较高，注意休息"
IF 某方向<10min/天 → "建议平衡三个方向的学习时间"
```

### 基于产出的建议
```
IF 笔记=0 连续 3 天 → "建议开始记录学习笔记"
IF 实操=0 连续 3 天 → "学习需要动手实践"
IF 作品=0 连续 7 天 → "尝试完成一个小项目"
```

const weekNum = Math.ceil((today - startDate) / (7 * 24 * 60 * 60 * 1000));
