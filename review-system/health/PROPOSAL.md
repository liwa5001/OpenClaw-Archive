## 1. 目标制定

### 1.1 健康堡核心目标

**使命：** 帮助用户科学管理健康，实现运动、饮食、睡眠的平衡发展

**三大支柱：**
- 💪 **运动表现** - 提升体能，科学训练
- 🥗 **营养健康** - 合理饮食，体重管理
- 😴 **睡眠恢复** - 优质睡眠，充分恢复

---

### 1.2 目标制定框架（与用户共同决定）

#### A. 运动目标（骑行专项）

| 周期 | 目标类型 | 示例目标 | 追踪指标 |
|------|----------|----------|----------|
| **周** | 训练量 | 骑行 150km | 距离、时长 |
| **月** | 能力提升 | FTP 提升 5% | FTP、平均功率 |
| **季** | 赛事准备 | 完成 100km 骑行 | 完赛时间、平均心率 |
| **年** | 综合成就 | 骑行 5000km | 总距离、总爬升 |

#### B. 体重/饮食目标

| 周期 | 目标类型 | 示例目标 | 追踪指标 |
|------|----------|----------|----------|
| **周** | 饮食习惯 | 16+8 执行 5 天 | 断食天数、热量摄入 |
| **月** | 体重变化 | 减重 2kg | 体重、体脂率 |
| **季** | 身体成分 | 体脂率降至 15% | 体脂率、肌肉量 |
| **年** | 健康状态 | 维持健康体重 | BMI、腰围、体检指标 |

#### C. 睡眠目标

| 周期 | 目标类型 | 示例目标 | 追踪指标 |
|------|----------|----------|----------|
| **周** | 睡眠时长 | 平均 7.5h/晚 | 平均时长、规律性 |
| **月** | 睡眠质量 | 深睡占比 20% | 深睡%、REM%、觉醒次数 |
| **季** | 作息规律 | 23:00 前入睡 80% | 入睡时间、起床时间 |
| **年** | 长期健康 | 睡眠评分 85+ | 综合睡眠评分 |

## 7. 分析方法

### 7.1 运动数据分析

**核心指标：**
- **训练负荷（TSS/CTL）** - 量化训练压力
- **心率区间分布** - 评估训练强度
- **功率曲线** - 能力变化趋势
- **恢复时间** - 避免过度训练

**分析方法：**
```
1. 周训练量趋势 → 判断是否超量恢复
2. 心率漂移分析 → 评估有氧能力
3. 功率体重比 → 爬坡能力预测
4. 训练一致性 → 与表现相关性
```

---

### 7.2 饮食/体重分析

**核心指标：**
- **热量平衡** - 摄入 vs 消耗
- **体重变化率** - 健康减重 0.5-1kg/周
- **体脂率趋势** - 肌肉保持情况
- **饮食一致性** - 习惯养成度

**分析方法：**
```
1. 周平均体重 vs 热量摄入 → 验证 TDEE 估算
2. 训练日 vs 休息日摄入 → 营养周期化
3. 断食执行率 vs 体重变化 → 16+8 效果评估
```

---

### 7.3 睡眠分析

**核心指标：**
- **睡眠效率** - 睡着时间/卧床时间（目标>85%）
- **深睡占比** - 身体恢复（目标 15-25%）
- **REM 占比** - 心理恢复（目标 20-25%）
- **睡眠规律性** - 入睡/起床时间标准差

**分析方法：**
```
1. 睡眠时长 vs 次日训练表现 → 确定个人最佳时长
2. 入睡时间 vs 深睡占比 → 优化作息
3. 运动时间 vs 睡眠质量 → 找到最佳训练时段
4. 咖啡因/酒精摄入 vs 睡眠 → 识别影响因素
```

---

### 7.4 跨领域关联分析

```
关联性分析示例：

1. 睡眠时长 vs 次日训练功率
   → 发现：睡眠<7h 时，功率下降 10%
   → 建议：保证 7.5h+ 睡眠

2. 热量摄入 vs 训练恢复
   → 发现：训练日摄入<2000kcal 时，恢复慢
   → 建议：训练日增加 300-500kcal

3. 断食执行 vs 体重变化
   → 发现：执行率>80% 时，月减重 2kg+
   → 建议：保持当前断食频率
```
