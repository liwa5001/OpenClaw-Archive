## 🔢 卡路里计算逻辑

### 食物识别关键词
```
主食类：
- 米饭/饭 → 130 kcal/100g
- 面条/面 → 112 kcal/100g
- 馒头/包子 → 223 kcal/100g
- 面包 → 265 kcal/100g

蛋白质：
- 鸡胸肉 → 165 kcal/100g
- 牛肉 → 250 kcal/100g
- 猪肉 → 143 kcal/100g
- 鱼 → 200 kcal/100g (平均)
- 鸡蛋 → 155 kcal/个

蔬菜：
- 青菜/蔬菜 → 25 kcal/100g (平均)
- 土豆 → 77 kcal/100g

菜品：
- 宫保鸡丁 → 450 kcal/份
- 鱼香肉丝 → 420 kcal/份
- 麻婆豆腐 → 350 kcal/份
- 红烧肉 → 600 kcal/份
- 清蒸鱼 → 250 kcal/份
```

### 份量估算
```
- 1 碗米饭 = 200g
- 1 个包子 = 100g
- 1 份菜 = 200-300g
- 1 杯豆浆 = 250ml
- 1 个鸡蛋 = 50g
```

## 📊 自动计算指标

### 1. 总卡路里
```
总卡路里 = 早餐 + 午餐 + 晚餐
```

### 2. 进食窗口
```
第一餐时间 = 早餐时间
最后一餐时间 = 晚餐时间
进食窗口时长 = 最后一餐 - 第一餐
断食时长 = 24 - 进食窗口
```

### 3. 16+8 判断
```
断食时长 ≥ 16h → ✅ 达标
断食时长 < 16h → ❌ 未达标
```

### 4. 睡眠时长
```
睡眠时长 = 起床时间 - 入睡时间
（如跨天，+24h 计算）
```

## 💡 建议生成逻辑

### 运动建议
```
IF 连续 3 天无训练 → "建议安排一次恢复性运动"
IF 连续 7 天高强度 → "建议安排 1-2 天休息"
IF 今日训练疲惫 → "明日建议 Z2 有氧或休息"
```

### 饮食建议
```
IF 卡路里>目标 +500 → "今日热量超标，明日可适当减少"
IF 蛋白质不足 → "建议增加蛋白质摄入（鸡胸肉/鱼/蛋）"
IF 蔬菜不足 → "建议增加蔬菜摄入"
IF 16+8 未达标 → "明日尝试提前晚餐时间"
```

### 睡眠建议
```
IF 睡眠<7h → "今晚尝试提前 30 分钟入睡"
IF 入睡>24:00 → "建议建立固定睡前仪式"
IF 质量<3 星 → "避免睡前使用电子设备"
```
