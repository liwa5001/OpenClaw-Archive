# 🏰 城堡六堡 (Castle Six) - 人生复盘系统

**创建日期：** 2026-03-08  
**版本：** v0.1 (MVP 阶段)  
**状态：** 🚧 开发中

---

## 📋 项目概述

「城堡六堡」是一个基于 sub-agent 的人生复盘系统，通过 6 个垂直领域的复盘 agent + 1 个综合汇总 agent，帮助用户全面记录和分析生活的方方面面。

---

## 🏗️ 架构设计

```
🏰 总复盘堡 (Summary)
│
├── 💼 事业堡 (Work)
├── 💪 健康堡 (Health) ← MVP 起点
├── 💕 关系堡 (Relationship)
├── 💰 财富堡 (Wealth)
├── 📚 成长堡 (Growth)
└── 🎮 生活堡 (Life)
```

---

## 📁 目录结构

```
/workspace/
├── agents/review-system/        # 共享配置
│   ├── PROJECT.md               # 项目说明（本文件）
│   ├── common/                  # 通用配置
│   ├── health/                  # 健康堡配置
│   ├── work/                    # 事业堡配置
│   ├── relationship/            # 关系堡配置
│   ├── wealth/                  # 财富堡配置
│   ├── growth/                  # 成长堡配置
│   ├── life/                    # 生活堡配置
│   └── summary/                 # 总复盘堡配置
│
├── daily-output/                # 每日输出
├── monthly-output/              # 每月输出
├── quarterly-output/            # 每季输出
└── yearly-output/               # 每年输出
```

---

## 🎯 六大支柱定义

| 堡名 | 核心问题 | 周期 | 输出目录 |
|------|----------|------|----------|
| 💼 事业堡 | 今天工作有进展吗？ | 日/月/季/年 | work-output/ |
| 💪 健康堡 | 今天身体状态如何？ | 日/月/季/年 | health-output/ |
| 💕 关系堡 | 今天和谁连接了？ | 日/月/季/年 | relationship-output/ |
| 💰 财富堡 | 今天钱花哪了？ | 日/月/季/年 | wealth-output/ |
| 📚 成长堡 | 今天学到了什么？ | 日/月/季/年 | growth-output/ |
| 🎮 生活堡 | 今天开心吗？ | 日/月/季/年 | life-output/ |
