## 🔄 配置变更流程

### 修改前
- [ ] 搜索相关配置文件（`grep -r "关键词" scripts/ memory/`）
- [ ] 列出所有相关文件
- [ ] 确认是否有旧配置需要清理

### 修改后
- [ ] 更新主要配置文件
- [ ] 删除或注释旧配置
- [ ] 更新 memory/YYYY-MM-DD.md 记录变更
- [ ] **在 cron 环境中测试命令路径**（`which 命令`）
- [ ] 更新 HEARTBEAT.md 或相关文档

### 验证
- [ ] 手动执行一次脚本测试
- [ ] 检查日志输出
- [ ] 确认目标渠道收到消息
- [ ] 记录验证结果到 memory 文件
