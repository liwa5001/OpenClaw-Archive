# 💪 健康堡 - 互动版每日问卷 (v3)

**版本：** v3.0 (互动增强版)  
**生效日期：** 2026-03-10  
**发送时间：** 每日 21:30

---

## 🎯 设计原则

1. **回复简单** - 大部分问题用数字/选项回复
2. **格式清晰** - 一眼看懂要填什么
3. **容错性强** - 忘记填某项也能处理
4. **即时反馈** - 回复后自动计算并确认

## 🔢 回复解析逻辑

### 运动解析
```
A0 → 休息日，运动评分=0
A1, 骑行，90, 3 → 
  - 类型：骑行
  - 时长：90 分钟
  - 强度：3 分 (中等)
  - 估算卡路里：90 × 8 × 体重 (kg) / 60 ≈ 850 kcal
```

### 饮食解析
```
B1 → 清淡，估算 1500 kcal
B2 → 正常，估算 1800 kcal
B3 → 油腻，估算 2200 kcal

详细回复 → 关键词匹配计算
```

### 睡眠解析
```
C1 → 1 分，睡眠质量差
C2 → 2 分，睡眠质量一般
C3 → 3 分，睡眠质量不错
C4 → 4 分，睡眠质量很好
C5 → 5 分，睡眠质量完美

带数据 → 记录到追踪表
```
